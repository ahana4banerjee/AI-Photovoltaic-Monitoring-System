# Database Documentation

## Overview

The project uses **SQLite** as the local historical database for storing all incoming telemetry generated by the simulated photovoltaic monitoring system.

The database enables:

- Historical analysis
- Trend visualization
- AI model training
- Fault investigation
- Predictive maintenance

---

pv_monitoring.db

- Output Power

- Panel Efficiency
- Health Score
